# 通訊協定說明
一般以 WebSocket 指定 `chatv1` 通訊協定連線，
管理用則以 `adminv1` 通訊協定連線(忽略人數上限)，
具體傳輸內容見 `ActionGroup.md`。

註冊、忘記密碼操作以 HTTP 進行，具體方法見 `HTTP.md`。

# 斷線狀態碼說明
* `1000` 正常登出
* `1000` 登出成功
* `1001` 伺服器關閉中
* `1008` 疑似攻擊或IP被封鎖，拒絕連線
* `4000` 伺服器維護中
* `4001` 伺服器鎖定中
* `4002` 伺服器超載或達人數上限
* `4003` 伺服器錯誤(暫定)
* `4100` 登入超時
* `4101` 帳號停用中
* `4102` 帳號或密碼錯誤
* `4103` 重複登入(拒絕登入)
* `4104` 被管理員踢出伺服器
